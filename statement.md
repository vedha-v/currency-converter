# Problem Statement
People such as tourists, students studying abroad, and small business owners often need precise, current financial information in the modern worldwide economy to make wise judgments. Depending on manual computations or static exchange rate tables causes major mistakes and unneeded financial risk. To give current exchange rate conversions between significant global currencies, a quick, dependable, and accessible digital tool is needed.

# Scope of the Project
1.The project is a Command-Line Interface (CLI) Live Currency Converter designed to:

2.Fetch and utilize real-time exchange rates via an external web API.

3.Provide conversion functionality for any two currency codes supported by the API (e.g., USD, EUR, INR, JPY, GBP).

4.Include core financial utility modules, such as saving favorite conversions and comparing historical rates, to meet the three major functional modules requirement.

5.Ensure robustness through comprehensive error handling for network failures, invalid currency codes, and non-numeric inputs.

The project does not include a Graphical User Interface (GUI) or user authentication.

# Target Users
The primary users of this application are:

1.International Travelers: Individuals needing quick, accurate conversions while abroad.

2.Students/Academics: Users needing to convert tuition fees, research grants, or stipends between currencies.

3.Casual Investors & Small Business Owners: Individuals needing to track or forecast small, non-commercial transactions.

# High-Level Features

Live Conversion Engine: Converts any amount between two standard currency codes using real-time data fetched from an external API.
